Matrix Multiplication
This is a simple Java console program that multiplies two matrices entered by the user.

Features:
  Takes matrix dimensions as input
  Checks if matrices are compatible for multiplication
  Accepts matrix elements from the user
  Performs matrix multiplication using nested loops
  Displays the resulting matrix
  
Purpose:
This program is useful for learning:
  2D arrays in Java
  Nested loops
  Basic matrix operations

PSEUDOCODE
START
Create Matrix A
Create Matrix B
Display Matrix A
Display Matrix B
For each row in Matrix A
For each row in Matrix B 
SET result to 0
For each number in the row
MULTIPLY and ADD to result
END FOR
DISPLAY new matrix
END
